# GetNextLine
_This program written in C allows to read into a file line by line_

## Build & Launch

```bash
git clone https://github.com/remyft/GNL
cd GNL && make -C libft
gcc -o GNL main.c get_next_line.c libft/libft.a && ./GNL main.c
```

## New method learned

* Static variables
* Simple linked list
* Memory reallocation
